# **App Name**: EnergySync

## Core Features:

- Dashboard Overview: Displays the user's current social energy level, biometric data, and upcoming scheduled events, to inform them of their current state.
- Activity Logging: Allows users to log activities (social, work, recharge) to track energy impact.
- Personalized Recharge Suggestions: AI-powered recommendations for recharge activities, suggesting personalized activities with impact scores, using an LLM tool to reason about the content and display options.
- Weekly Report: Provides a weekly report of total energy drained and recharged, highlighting the week's most valuable recharge activity.
- External Data Sync: Ability to sync external calendar and health data, to automatically display events or update data.
- Community Mode: Switch to share an anonymized status update within a community setting.

## Style Guidelines:

- Primary color: Deep violet (#673AB7) for a calming and sophisticated feel.
- Background color: Light lavender (#E1DDF7) to provide a soft and clean backdrop.
- Accent color: Soft periwinkle (#9FA8DA) for subtle highlighting of key elements.
- Body and headline font: 'Inter' sans-serif font for clear, modern readability.
- Use minimalist, outlined icons to represent different activities and categories, matching the general feel.
- Employ a card-based layout with rounded corners and subtle shadows for an organized, intuitive interface.
- Implement smooth transitions and loading animations to enhance user engagement and provide feedback.
